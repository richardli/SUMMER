# Simulated child mortality person-month dataset.

A small simulated dataset with 4 regions and 5 survey years. This does
not represent any real country's data and are based on a subset of the
model dataset provided by DHS.

## Usage

``` r
data(DemoData)
```

## Format

A list of with five components, named by survey year.

# Simulated dataset for prevalence mapping.

A small fake dataset with 8 regions and two response variables: age and
tobacco.use. This does not represent any real country's data and are
based on a subset of the model dataset provided by DHS.

## Usage

``` r
data(DemoData2)
```

## Format

A data.frame of 7 variables.

## Source

<https://dhsprogram.com/data/model-datasets.cfm>

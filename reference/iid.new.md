# New random IID models for m-year to period random effects

New random IID models for m-year to period random effects

## Usage

``` r
iid.new(
  cmd = c("graph", "Q", "mu", "initial", "log.norm.const", "log.prior", "quit"),
  theta = NULL
)
```

## Arguments

- cmd:

  list of model components

- theta:

  log precision

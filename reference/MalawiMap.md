# Malawi Admin-2 map

SpatialPolygonsDataFrame objects that reflect the Admin 2 regions in
Malawi, including the Likoma island. The Admin 2 region names are in the
ADM2_EN field.

## Usage

``` r
data(MalawiMap)
```

## Format

An object of class `SpatialPolygonsDataFrame` with 28 rows and 14
columns.
